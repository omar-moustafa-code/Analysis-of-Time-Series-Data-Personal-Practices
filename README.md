# Analysis of Time Series Data - Personal Practices

## Overview
This repository contains my personal practice exercises for the "Analysis of Time Series Data" course. The primary goal was to reinforce understanding of key time series concepts and techniques by implementing them in R. The exercises cover fundamental methods for analyzing and forecasting time-dependent data.

## Key Techniques Practiced
The practices focus on core time series analysis methods, including:
*   **Moving Averages**: Smoothing data to identify trends.
*   **Filters**: Applying various filters to extract signals from noise.
*   **Decompositions**: Breaking down a time series into its trend, seasonal, and residual components.

## How to Use This Repository
1.  Clone the repository to your local machine.
2.  To view the practice summaries, open the `.pdf` files.
3.  To run the R code, open the `.Rmd` files in RStudio or the `.ipynb` files in Jupyter (with an R kernel installed).
4.  Ensure you have the necessary R packages installed (e.g., `forecast`, `tseries`, `astsa`).
